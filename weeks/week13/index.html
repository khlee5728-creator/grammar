Learning Objectives

prefer + -ing / to...

prefer + 동명사 / 부정사

Two correct patterns with prefer

prefer + -ing  /  prefer + to + verb

prefer A to B  (with -ing)  |  prefer to do A rather than (do) B  (with to)

prefer A to B (동명사) | prefer to do A rather than (do) B (부정사)

"I prefer driving to taking the train."
prefer + -ing + to + -ing prefer + 동명사 + to + 동명사
"I prefer to drive rather than take the train."
prefer + to + verb + rather than + base form prefer + to부정사 + rather than + 원형
"Would you prefer coffee or tea?" — "Coffee, please."
would prefer + noun (for specific situations) 구체적 상황에서 would prefer + 명사

would rather + base form

would rather + 동사원형

Important: would rather is followed by the BASE FORM

Subject + would rather + base form (NOT to)

Negative: I'd rather not + base form  |  Comparison: I'd rather do A than do B

부정: I'd rather not + 원형 | 비교: I'd rather A than B

"Should we take the train?" — "I'd rather drive." (not to drive)
would rather + base form would rather + 동사원형 (to 없음!)
"I'd rather stay home tonight than go to the movies."
I'd rather + base ... than + base 영화관보다 집에 있겠다
"I'm tired. I'd rather not go out tonight, if you don't mind."
Negative: I'd rather not + base form 부정: I'd rather not + 동사원형
"Who's going to drive?" — "I'd rather you drove."
I'd rather somebody did something (past subjunctive for others) 다른 사람에게: I'd rather + 주어 + 과거형

📘 GIU Unit 57, p.114-115

Preposition + -ing

전치사 + 동명사

After a preposition, always use -ing (NEVER the base form)

preposition (in, for, about, at, of, with, instead of, in spite of) + -ing

This is one of the most important rules in English grammar!

이것은 영문법에서 가장 중요한 규칙 중 하나입니다!

"Are you interested in working for us?"
in + -ing 우리를 위해 일하는 것에 관심이 있나요?
"Thanks very much for inviting me to your party."
for + -ing 파티에 초대해 주셔서 감사합니다
"What are the advantages of having a car?"
of + -ing 차를 갖는 것의 장점은?
"Amy went to work in spite of feeling sick."
in spite of + -ing 아파도 불구하고 출근했다

📘 GIU Unit 58, p.116-117

before / after / by / without + -ing

before / after / by / without + 동명사

"Before to go out, I called Sarah."
WRONG: preposition + to + verb 틀림: 전치사 뒤에 to + 동사원형 불가
"Before going out, I called Sarah."
CORRECT: before + -ing 맞음: before + 동명사
"You can improve your English by reading more."
by + -ing (method / means) by + 동명사 (방법/수단)
"We ran ten miles without stopping."
without + -ing 멈추지 않고 10마일을 달렸다
"What did you do after leaving school?"
after + -ing 학교를 떠난 후 무엇을 했나요?

📘 GIU Unit 58, p.116-117

to + -ing (look forward to doing)

to가 전치사일 때 + 동명사

Tricky! "to" can be a preposition (not just part of the infinitive)

look forward to / prefer ... to / be used to + -ing

When "to" is a preposition (like in/for/about), the verb after it must be -ing!

"to"가 전치사일 때 (in/for/about처럼), 뒤에 오는 동사는 반드시 -ing!

"Are you looking forward to go on vacation?"
WRONG: "to" here is a preposition, not an infinitive marker 틀림: 여기서 to는 전치사!
"Are you looking forward to going on vacation?"
CORRECT: look forward to + -ing 맞음: look forward to + 동명사
"I prefer driving to taking the train."
prefer ... to + -ing (to = preposition) prefer ... to + 동명사 (to = 전치사)
"I'm fed up with traveling by train."
fed up with + -ing 기차 여행에 질렸다

📘 GIU Unit 58, p.116-117

used to do vs be used to doing

used to do (과거 습관) vs be used to doing (익숙함)

used to + base form

Past habit or state (no longer true)

과거 습관 (지금은 아님)

"I used to have a car, but I sold it."
Past state, no longer true. 과거 상태, 지금은 아님

be used to + -ing / noun

Accustomed to / familiar with

~에 익숙하다

"John lives alone. He's used to living alone."
He is accustomed to it. 혼자 사는 것에 익숙하다
"Rachel is American but lives in Tokyo. At first she wasn't used to driving on the left."
wasn't used to + -ing = it was strange/unfamiliar 왼쪽 운전에 익숙하지 않았다

📘 GIU Unit 59, p.118-119

get used to + -ing

~에 익숙해지다

"get used to" = become accustomed to (process)

Subject + get/got used to + -ing / noun

"be used to" = already accustomed. "get used to" = the process of becoming accustomed.

"be used to" = 이미 익숙함. "get used to" = 익숙해지는 과정.

"After a while, Rachel got used to driving on the left."
She eventually became accustomed to it. 결국 왼쪽 운전에 익숙해졌다
"Now it's no problem. She is used to driving on the left."
She is accustomed now (result). 이제 익숙한 상태 (결과)
"Lauren has to get up at 6:30. She isn't used to getting up so early."
She is not accustomed to it (yet). 아직 그렇게 일찍 일어나는 것에 익숙하지 않다

📘 GIU Unit 59, p.118-119

Verb + Object + to...

동사 + 목적어 + to 부정사

Some verbs follow the pattern: verb + object + to + verb

want / ask / tell / expect / advise / allow / encourage + someone + to do

The object (person) comes between the verb and "to + verb".

목적어(사람)가 동사와 "to + 동사" 사이에 옵니다.

"I want you to be careful."
want + object + to + verb want + 목적어 + to + 동사
"He told me to wait outside."
tell + object + to + verb tell + 목적어 + to + 동사
"The doctor advised her to rest."
advise + object + to + verb advise + 목적어 + to + 동사

Verb + Preposition + -ing

동사 + 전치사 + 동명사

Many verbs are followed by a preposition + -ing

dream of / feel like / insist on / succeed in / think of + -ing
"I don't feel like going out tonight."
feel like + -ing 오늘 밤 나가고 싶지 않다
"They insisted on paying for the meal."
insist on + -ing 식사비를 내겠다고 고집했다
"Has Rob succeeded in finding a job yet?"
succeed in + -ing 직장을 찾는 데 성공했나요?

Verb + Object + Preposition + -ing

"He accused me of telling lies."
accuse ... of + -ing 거짓말한다고 비난했다
"The rain didn't stop us from enjoying our trip."
stop ... from + -ing 비가 와도 여행을 즐기는 것을 막지 못했다
"I thanked everyone for helping me."
thank ... for + -ing 도와줘서 모두에게 감사했다

Expressions + -ing

표현 + 동명사

Common expressions followed by -ing

there's no point in / it's worth / have trouble / spend time + -ing
"There's no point in having a car if you never use it."
there's no point in + -ing 사용하지 않으면 차를 가지고 있는 의미가 없다
"It's a great movie. It's worth seeing."
it's worth + -ing 볼 가치가 있다
"I had no trouble finding a place to stay."
have trouble + -ing (not "trouble to find") 숙소를 찾는 데 어려움이 없었다
"He spent hours trying to repair the clock."
spend time + -ing 시계를 고치려고 몇 시간을 보냈다
"I waste a lot of time doing nothing."
waste time + -ing 아무것도 안 하면서 시간을 낭비한다
"We go swimming every weekend."
go + -ing (for sports/activities: go sailing, go fishing, go skiing...) go + 동명사 (스포츠/활동)

Common Mistakes for Korean Speakers

한국어 화자가 자주 하는 실수

⚠ L1 Interference: "would rather" + "to"

Korean speakers often add "to" after "would rather" because "to" is common in other infinitive patterns.

다른 부정사 패턴에서 "to"를 많이 쓰기 때문에 "would rather" 뒤에도 "to"를 추가하는 실수를 합니다.

"I'd rather to stay home."
"I'd rather stay home."
⚠ L1 Interference: Confusing "used to do" and "be used to doing"

Korean has one expression for both meanings, making this distinction difficult.

한국어에서는 하나의 표현으로 두 의미를 나타내므로 구분이 어렵습니다.

"I'm used to get up early."
Wrong! "be used to" must be followed by -ing 틀림! "be used to" 뒤에는 반드시 -ing
"I'm used to getting up early."
Correct: be used to + -ing (accustomed to) 맞음: be used to + -ing (익숙하다)
