-ing and to... Part 2

동명사와 부정사 (2)

Units 57 – 61

Learning Objectives

이번 주 학습 목표

  • Use prefer and would rather correctly with -ing, to..., and base form prefer와 would rather를 -ing, to..., 원형과 올바르게 사용한다
  • Master the rule: preposition + -ing (never preposition + to + base form) 전치사 + -ing 규칙을 익힌다 (전치사 뒤에 to + 원형 불가)
  • Distinguish be/get used to doing from used to do be/get used to doing과 used to do의 차이를 구분한다
  • Use verb + object + to... pattern correctly 동사 + 목적어 + to... 패턴을 올바르게 사용한다

prefer + -ing / to...

prefer + 동명사 / 부정사

Two correct patterns with prefer

prefer + -ing  /  prefer + to + verb

Both forms are grammatically correct. The -ing form is slightly more common in British English.

두 형태 모두 문법적으로 올바릅니다.

"I prefer staying home on weekends."
prefer + -ing prefer + 동명사
"I prefer to stay home on weekends."
prefer + to + verb prefer + to + 동사원형

would rather + base form

would rather + 동사원형

Important: would rather is followed by the BASE FORM

Subject + would rather + base form (NOT to)

"Would rather" = "I'd rather" (contraction). It is NEVER followed by "to".

"Would rather"는 절대 "to"와 함께 사용하지 않습니다!

"I'd rather stay home tonight."
would rather + base form would rather + 동사원형
"I'd rather to stay home tonight."
WRONG! Never use "to" after "would rather". 틀림! would rather 뒤에 to 사용 불가
"She'd rather not go out."
Negative: would rather not + base form 부정: would rather not + 동사원형

Preposition + -ing

전치사 + 동명사

After a preposition, always use -ing (NEVER the base form)

preposition (after, before, without, instead of, by, for, about, in, at, of) + -ing

This is one of the most important rules in English grammar!

이것은 영문법에서 가장 중요한 규칙 중 하나입니다!

"She left without saying goodbye."
"I'm interested in learning Japanese."
"She's good at singing."
"Instead of studying, he played games."

Common Preposition + -ing Errors

전치사 + 동명사 빈출 오류

"She left without to say goodbye."
WRONG: preposition + to + verb 틀림: 전치사 뒤에 to + 동사원형 불가
"She left without saying goodbye."
CORRECT: preposition + -ing 맞음: 전치사 + 동명사
"I'm thinking about to change my job."
"I'm thinking about changing my job."
"Before to leave, check your bag."
"Before leaving, check your bag."

used to do vs be used to doing

used to do (과거 습관) vs be used to doing (익숙함)

used to + base form

Past habit or state (no longer true)

과거 습관 (지금은 아님)

"I used to get up late."
In the past, but not anymore. 과거에는 그랬지만 지금은 아님
"She used to live in Busan."
She lived there before, but not now. 전에는 부산에 살았지만 지금은 아님

be used to + -ing

Accustomed to / familiar with

~에 익숙하다

"I'm used to getting up early."
I am accustomed to it now. 지금 그것에 익숙하다
"She's used to living alone."
She is familiar with it. 그녀는 그것에 익숙하다

get used to + -ing

~에 익숙해지다

"get used to" = become accustomed to (process)

Subject + get/got used to + -ing

"be used to" = already accustomed. "get used to" = the process of becoming accustomed.

"be used to" = 이미 익숙함. "get used to" = 익숙해지는 과정.

"I'm slowly getting used to waking up early."
The process of becoming accustomed. 익숙해지는 과정
"He got used to driving on the left in the UK."
He eventually became accustomed to it. 결국 그것에 익숙해졌다

Verb + Object + to...

동사 + 목적어 + to 부정사

Some verbs follow the pattern: verb + object + to + verb

want / ask / tell / expect / advise / allow / encourage + someone + to do

The object (person) comes between the verb and "to + verb".

목적어(사람)가 동사와 "to + 동사" 사이에 옵니다.

"I want you to be careful."
want + object + to + verb want + 목적어 + to + 동사
"He told me to wait outside."
tell + object + to + verb tell + 목적어 + to + 동사
"The doctor advised her to rest."
advise + object + to + verb advise + 목적어 + to + 동사

More Verb + Object + to... Examples

추가 예문

"She asked me to help her."
ask + object + to + verb ask + 목적어 + to + 동사
"They expected us to arrive on time."
"My parents allowed me to go to the party."
"The teacher encouraged the students to speak English."

Common Mistakes for Korean Speakers

한국어 화자가 자주 하는 실수

⚠ L1 Interference: "would rather" + "to"

Korean speakers often add "to" after "would rather" because "to" is common in other infinitive patterns.

다른 부정사 패턴에서 "to"를 많이 쓰기 때문에 "would rather" 뒤에도 "to"를 추가하는 실수를 합니다.

"I'd rather to stay home."
"I'd rather stay home."
⚠ L1 Interference: Confusing "used to do" and "be used to doing"

Korean has one expression for both meanings, making this distinction difficult.

한국어에서는 하나의 표현으로 두 의미를 나타내므로 구분이 어렵습니다.

"I'm used to get up early."
Wrong! "be used to" must be followed by -ing 틀림! "be used to" 뒤에는 반드시 -ing
"I'm used to getting up early."
Correct: be used to + -ing (accustomed to) 맞음: be used to + -ing (익숙하다)
