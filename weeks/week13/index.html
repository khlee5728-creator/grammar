prefer and would rather Unit 57A-B

prefer + 동명사 / 부정사

Two correct patterns with prefer

📖 "When you say what you prefer in general, you can use 'prefer to ...' or 'prefer -ing'." (Unit 57A)

prefer + -ing  /  prefer + to + verb

prefer A to B  (with -ing)  |  prefer to do A rather than (do) B  (with to)

prefer A to B (동명사) | prefer to do A rather than (do) B (부정사)

"I prefer driving to taking the train."
prefer + -ing + to + -ing
"I prefer to drive rather than take the train."
prefer + to + verb + rather than + base form
"Would you prefer coffee or tea?" — would prefer + noun (specific situations)

📖 GIU Unit 57, p.114-115

would rather + base form Unit 57C-D

would rather + 동사원형

would rather is followed by the BASE FORM (NOT to)

Subject + would rather + base form

Negative: I'd rather not + base  |  Comparison: I'd rather do A than do B

"I'd rather drive." (not to drive) — would rather + base form
"I'd rather stay home than go to the movies."
"I'd rather you drove." — I'd rather sb did (past subjunctive)

📖 GIU Unit 57, p.114-115

Preposition (in/for/about, etc.) + -ing Unit 58A

전치사 + 동명사

After a preposition, always use -ing (NEVER the base form)

preposition (in, for, about, at, of, with, instead of, in spite of) + -ing
"Are you interested in working for us?" — in + -ing
"Thanks very much for inviting me to your party." — for + -ing
"Amy went to work in spite of feeling sick." — in spite of + -ing

📖 GIU Unit 58, p.116-117

Preposition + -ing — before / after / by / without Unit 58B

before / after / by / without + 동명사

"Before to go out, I called Sarah."
WRONG: preposition + to + verb 틀림: 전치사 뒤에 to + 동사원형 불가
"Before going out, I called Sarah."
CORRECT: before + -ing 맞음: before + 동명사
"You can improve your English by reading more."
by + -ing (method / means) by + 동명사 (방법/수단)
"We ran ten miles without stopping."
without + -ing 멈추지 않고 10마일을 달렸다

📖 GIU Unit 58, p.116-117

to + -ing (look forward to doing) Unit 58C

to가 전치사일 때 + 동명사

"to" can be a preposition → followed by -ing

look forward to / prefer ... to / be used to + -ing
"Are you looking forward to go on vacation?" — WRONG
"Are you looking forward to going on vacation?" — look forward to + -ing
"I prefer driving to taking the train." — prefer ... to + -ing (to = preposition)

📖 GIU Unit 58, p.116-117

be/get used to ... (I'm used to ...) Unit 59

used to do (과거 습관) vs be used to doing (익숙함)

used to + base form

Past habit (no longer true) 과거 습관 (지금은 아님)

"I used to have a car, but I sold it."

be used to + -ing / noun

Accustomed to ~에 익숙하다

"He's used to living alone."
"At first she wasn't used to driving on the left."
wasn't used to + -ing = unfamiliar 왼쪽 운전에 익숙하지 않았다

📖 GIU Unit 59, p.118-119

get used to + -ing Unit 59B

~에 익숙해지다

"get used to" = become accustomed to (process)

Subject + get/got used to + -ing / noun

"be used to" = already accustomed  |  "get used to" = becoming accustomed

"Rachel got used to driving on the left." — became accustomed
"She is used to driving on the left." — accustomed now (result)
"She isn't used to getting up so early." — not accustomed (yet)

동사 + 목적어 + to 부정사

verb + object + to + verb

want / ask / tell / expect / advise / allow / encourage + someone + to do
"I want you to be careful." — want + object + to
"He told me to wait outside." — tell + object + to

Verb + Preposition + -ing (succeed in -ing / insist on -ing) Unit 60A

동사 + 전치사 + 동명사

Many verbs are followed by a preposition + -ing

dream of / feel like / insist on / succeed in / think of + -ing

there's no point in -ing, it's worth -ing, etc. Unit 61A-C

표현 + 동명사

Common expressions followed by -ing

there's no point in / it's worth / have trouble / spend time + -ing
"There's no point in having a car if you never use it."
"It's worth seeing." — it's worth + -ing

spend time -ing / go -ing Unit 61D-E

표현 + 동명사 (계속)

"He spent hours trying to repair the clock."
spend time + -ing 시계를 고치려고 몇 시간을 보냈다
"I waste a lot of time doing nothing."
waste time + -ing 아무것도 안 하면서 시간을 낭비한다
"We go swimming every weekend."
go + -ing (for sports/activities: go sailing, go fishing, go skiing...) go + 동명사 (스포츠/활동)

📖 GIU Unit 61, p.122-123

Common Mistakes for Korean Speakers

한국어 화자가 자주 하는 실수

⚠ L1 Interference: "would rather" + "to"

Korean speakers often add "to" after "would rather" because "to" is common in other patterns.

"I'd rather to stay home." → "I'd rather stay home."
⚠ L1 Interference: Confusing "used to do" and "be used to doing"

Korean has one expression for both meanings, making this distinction difficult.

"I'm used to get up early." → "I'm used to getting up early."
"be used to" + -ing (not base form) "be used to" 뒤에는 반드시 -ing

Week 13 Summary

  • prefer: + -ing or + to + verb (both OK); prefer A to B / prefer to do rather than do
  • would rather: + base form (NEVER "to"); I'd rather somebody did
  • Preposition + -ing: without stopping, by reading, in spite of feeling
  • "to" as preposition: look forward to doing, prefer ... to doing
  • used to + base: past habit / be/get used to + -ing: accustomed to
  • Verb + prep + -ing: feel like doing, insist on doing, succeed in doing
  • Expressions + -ing: it's worth doing, have trouble doing, spend time doing
