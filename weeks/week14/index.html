이번 주 학습 목표

  • Distinguish for + -ing (purpose of a thing) from to + verb (person's purpose) for + -ing (사물의 용도)와 to + 동사 (사람의 목적)를 구분한다

to... for Purpose

목적을 나타내는 to...

We use to + verb to say WHY somebody does something

Subject + verb ... + to + verb (= in order to / so as to)

"to + verb", "in order to + verb", and "so as to + verb" all express purpose. "in order to" and "so as to" are more formal.

"to + 동사", "in order to", "so as to" 모두 목적을 나타냅니다. in order to와 so as to가 더 격식체입니다.

"I went to the store to buy milk."
Purpose: to buy milk. 목적: 우유를 사기 위해
"I went to the store in order to buy milk."
Same meaning, more formal. 같은 의미, 더 격식체
"She whispered so as not to wake the baby."
Negative purpose: so as not to / in order not to 부정 목적: ~하지 않기 위해

for + noun / for + -ing

사물의 용도: for + 명사 / for + -ing

Use "for + -ing" to describe what something is USED for

"What is this for?" → "It's for cutting paper."

Important: Do NOT use "for + to + verb" for purpose. Use "to + verb" or "for + -ing".

중요: 목적을 나타낼 때 "for + to + 동사"를 사용하지 마세요.

"This button is for to open the door."
WRONG: "for + to + verb" is not correct. 틀림: "for + to + 동사"는 불가
"This button is for opening the door."
for + -ing (purpose of the thing) for + -ing (사물의 용도)
"This button is to open the door."
to + verb (also correct for purpose) to + 동사 (목적을 나타냄, 역시 가능)

Adjective + to...

형용사 + to 부정사

Many adjectives can be followed by to + verb

It's + adjective + to + verb

Common adjectives: easy, hard, difficult, impossible, nice, kind, good, important, dangerous

주요 형용사: easy, hard, difficult, impossible, nice, kind, good, important, dangerous

"It's easy to learn this grammar."
It + be + adjective + to + verb It + be + 형용사 + to + 동사
"She was kind to help us."
Person + be + adjective + to + verb 사람 + be + 형용사 + to + 동사
"This problem is difficult to solve."
Thing + be + adjective + to + verb 사물 + be + 형용사 + to + 동사

Perception Verbs: see/hear/watch + do/doing

지각동사: see/hear/watch + 원형/동명사

Two patterns with perception verbs

see/hear/watch + object + base form (complete action)
see/hear/watch + object + -ing (action in progress)

Base form = you saw the WHOLE action. -ing = you saw the action IN PROGRESS.

동사원형 = 전체 행동을 봄. -ing = 진행 중인 행동을 봄.

Perception Verbs: Examples

지각동사: 예문 비교

+ base form (complete)

동사원형 = 처음부터 끝까지

"I saw him cross the road."
I watched the whole action (from one side to the other). 한쪽에서 다른 쪽까지 전체를 봄
"I heard her sing a song."
I heard the complete song. 노래 전체를 들음

+ -ing (in progress)

-ing = 진행 중인 일부만

"I saw him crossing the road."
I saw him in the middle of crossing. 건너는 중간에 봄
"I heard her singing a song."
I heard part of the singing (in progress). 노래하고 있는 중에 들음

-ing Phrases (Participle Clauses)

-ing구 (분사구문)

-ing phrases give extra information about the subject

-ing phrase, + main clause

The -ing phrase and the main clause MUST have the same subject.

-ing구와 주절은 반드시 같은 주어를 가져야 합니다.

"Walking along the street, I met an old friend."
= While I was walking along the street, I met an old friend. = 길을 걷다가 옛 친구를 만났다
"Having finished dinner, we went out."
= After we had finished dinner, we went out. = 저녁을 먹은 후에 나갔다
"Not knowing what to do, she called for help."
= Because she didn't know what to do, she called for help. = 무엇을 해야 할지 몰라서 도움을 요청했다

More Participle Clause Patterns

분사구문 추가 패턴

Simultaneous actions

동시에 일어나는 행동

"Smiling broadly, she accepted the award."
She smiled AND accepted at the same time. 미소를 지으며 동시에 상을 받았다

Reason / Cause

이유/원인

"Being tired, I went to bed early."
= Because I was tired... = 피곤했기 때문에...

Completed earlier action (having + past participle)

먼저 완료된 행동 (having + 과거분사)

"Having read the book, I returned it to the library."
= After I had read the book... = 책을 다 읽은 후에...

Common Mistakes for Korean Speakers

한국어 화자가 자주 하는 실수

⚠ L1 Interference: Perception verbs do/doing distinction

Korean does not distinguish between "see someone do" and "see someone doing". Both translate the same way in Korean.

한국어는 "하는 것을 보다"로 동일하게 번역되어 구분이 어렵습니다.

"I saw him cross the road." = 전체 행동을 봄 (처음~끝)
"I saw him crossing the road." = 진행 중인 행동을 봄 (중간)
⚠ L1 Interference: Participle clauses feel unnatural

Korean uses different connective endings (-면서, -고, -아서) whereas English uses -ing phrases. Practice converting Korean connective patterns to English participle clauses.

한국어는 연결어미(-면서, -고, -아서)를 사용하지만 영어는 -ing구를 사용합니다.

"길을 걸으면서" → "Walking along the street, ..."
"뭘 해야 할지 몰라서" → "Not knowing what to do, ..."
