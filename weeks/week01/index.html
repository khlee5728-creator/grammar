How to Use the Textbook

교재 활용법

Left Page — Grammar Explanations

왼쪽 페이지: 문법 설명

  • Clear grammar rules with visual context
  • Example sentences showing real usage
  • Diagrams & timelines for tenses
  • Comparison with similar structures

Right Page — Exercises

오른쪽 페이지: 연습 문제

  • Practice exercises for each grammar point
  • Fill-in-the-blank & sentence completion
  • Error correction tasks
  • Check answers at the back of the book
  • 1. 왼쪽 페이지부터 읽으세요 — Read the left page explanation first before doing exercises.
  • 2. 예문을 소리 내어 읽으세요 — Read example sentences aloud for better retention.
  • 3. 연습 문제를 반드시 풀어보세요 — Always complete the right-page exercises; do not skip them.
  • 4. 틀린 문제는 표시해 두세요 — Mark difficult questions and review them before exams.
Show the textbook to students if possible. Explain the consistent left-page/right-page layout. Encourage students to always complete exercises in class or at home. The study tips here are based on effective language learning research.
